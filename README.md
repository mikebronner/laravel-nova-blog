# laravel-nova-blog
Drop-in Blog module for Laravel using Nova as an admin panel.

## Requirements

| Requirement | Version |
|---|---|
| PHP | 8.2, 8.3, 8.4, 8.5 |
| Laravel | 10.x, 11.x, 12.x |
| Laravel Nova | 4.x, 5.x |

## Compatible Plugins
- Laravel Governor
- Laravel Nova Categories

## Installation
1. `composer require genealabs/laravel-nova-blog`
