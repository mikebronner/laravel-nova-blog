# laravel-nova-blog
Drop-in Blog module for Laravel using Nova as an admin panel.

## Requirements
- PHP 8.2+
- Laravel 10.x, 11.x, 12.x, or 13.x
- Laravel Nova 4.x or 5.x

## Version Support

| Laravel | Package |
|---------|---------|
| 10.x   | ✅      |
| 11.x   | ✅      |
| 12.x   | ✅      |
| 13.x   | ✅      |

## Compatible Plugins
- Laravel Governor
- Laravel Nova Categories

## Installation
1. `composer require genealabs/laravel-nova-blog`
